# EASYMANAGE Warehouse Management System

## วิธีการติดตั้งและรันโปรเจกต์

**วิธีที่ 1: รันผ่าน Batch File**
1. ดับเบิลคลิกที่ไฟล์ `Start_Server.bat`
2. ระบบจะทำการรันเซิร์ฟเวอร์และเปิดหน้าเว็บเบราว์เซอร์ให้โดยอัตโนมัติ

**วิธีที่ 2: รันผ่าน Terminal (Command Line)**
1. เปิด Terminal ในโฟลเดอร์โปรเจกต์
2. ติดตั้ง Dependencies ทั้งหมดด้วยคำสั่ง: `npm install`
3. รันเซิร์ฟเวอร์ด้วยคำสั่ง: `npm run dev`
4. เปิดเบราว์เซอร์และไปที่: http://localhost:3000
